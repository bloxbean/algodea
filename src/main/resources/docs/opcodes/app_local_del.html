• Opcode: 0x68
  • Pops: ... stack, {uint64 A}, {[]byte B}
  • Pushes: None
  • delete from account specified by Txn.Accounts[A] local state key B of the current application
  • LogicSigVersion >= 2
  • Mode: Application

params: account index, state key.

Deleting a key which is already absent has no effect on the application local state. (In particular, it does not cause the program to fail.)
